# Airline Reservation System
 A simple console menu based flight reservation system
 
 This is a very simple system that has no login feature - meaning the user is not requires to login to book a flight. This assumes all flights fly out of Seattle. There are 7 different routes that a user can pick from. The user can book a ticket, reserve a seat, view flight details, ticket details & passenger information. The user can also cancel their ticket. It also takes the user through a payment process, where the user has to select whether to pay by Debit, Credit or bank account. once the user has booked a ticket, they can use the ticket number to view ticket details and passenger information. They can also cancel their ticket if they change theit mind.
 
 This system was developed using C++ Programming Language
